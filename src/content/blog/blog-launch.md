# 写在开始

这是 `Spark'Blog` 的第一篇文章。

这个站点暂时还没有堆很多内容，但它已经不再是一个保留模板示例数据的空壳，而是一个真正准备从自己的学习和项目记录开始慢慢长出来的博客。

## 这个博客会写什么

接下来，这里会主要记录三类内容：

1. Java 后端学习和项目实践。
2. Go 的入门、对比和踩坑记录。
3. 做项目、改博客、整理笔记时遇到的真实问题和复盘。

## 为什么要写下来

很多东西如果只停留在“我好像学过”，其实很快就会变成模糊印象。

把它们写成文章，一方面是为了逼自己把问题想清楚，另一方面也是为了以后回头看时，能知道自己到底是怎么一步一步走过来的。

## 从现在开始

这篇文章就当作一个起点。

后面的内容不一定一开始就很多，但会尽量保持真实、持续和有记录价值。
